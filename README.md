# Bookeeper
* 카카오 책 검색 API를 활용한 스프링 기반 웹애플리케이션
* 개인프로젝트 / 팀장
* 개발 기간 : 2021.04 ~ 2021.06

## 기술 스택
* Java 8, Spring Framework, Tomcat, Maven
* HTML, CSS, JS, BootStrap
* MySQL
* Kakao Book Search API
* AWS EC2, AWS RDS
